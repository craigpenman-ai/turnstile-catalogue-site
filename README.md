# Turnstile Archive — public catalogue

This is the **public-facing** static catalogue for the Turnstile Archive collection —
auto-generated, never edited by hand.

## How this gets updated

1. In the Turnstile Archive app, publish the items/bundles you want visible, then click
   **Export public catalogue** on the Dashboard. It writes straight into this folder.
2. Open **GitHub Desktop**, review the changes it picked up, write a short commit summary
   (e.g. "Add 12 more programmes"), and click **Commit to main** then **Push origin**.
3. GitHub Pages picks up the change automatically within a minute or two.

Nothing here is committed automatically — you're always in control of what goes live and when.

## What's deliberately excluded

Only items explicitly marked **published** in the app appear here. Box/bag locations, private
valuation notes, AI confidence scores, and internal notes never leave the private database.

This page is not password-protected — anyone with the link can view it — but it's marked
`noindex` so search engines won't list it. Treat the link as semi-private.
